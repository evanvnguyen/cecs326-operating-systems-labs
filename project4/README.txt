How to compile:
----------------
1. Ensure your system has the latest version of gcc installed.
2. cd into this folder.
3. Open terminal.
4. To compile fcfs, priority, and rr:
	-> make fcfs
	-> make priority
	-> make rr
5. Executing these commands will compile and create each .o executable.

How to run:
----------------
1. Type ./<executable> schedule.txt
	-> ./fcfs schedule.txt
	-> ./priority schedule.txt
	-> ./rr priority schedule.txt

Enjoy
